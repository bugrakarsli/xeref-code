## Product Positioning

Xeref Code is the IDE-native extension experience for Xeref. It brings the Xeref coding agent directly into the editor, and should remain clearly differentiated from the main web app — the IDE surface is for in-flow coding work, not a re-hosted version of the SaaS product — while complementing it as part of the broader Xeref platform.
